# linux中用户对文件的权限

  rw-   rw-   r--

（1）（2）（3）

（1）[u] 代表文件所有者对文件的使用权限

（2）[g]  代表文件所有组对文件的使用权限

（3）[o]  代表其他人对文件的使用权限

对权限的理解

r   可读（read)
 对文件：是否可以查看文件中的内容 

 对目录：是否查看目录中有什么子文件或者子目录

w   可写（write) 
  对文件：是否可以改变文件里面记录的字符

  对目录：是否可以对目录中子目录或子文件的元数据进行更改

x  可执行（execute)
  对文件：是否可以通过文件名称调用文件内记录的程序

  对目录：是否可以进入目录

| chmod |  u   |  +   |  r   | 文件或目录 |
| :---: | :--: | :--: | :--: | :--------: |
|       |  g   |  -   |  w   |            |
|       |  0   |  =   |  x   |            |

 rwx 的权限数分别为：r =4 , w=2, x=1
